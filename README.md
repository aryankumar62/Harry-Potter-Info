# 🧙‍♂️ Harry Potter Character Explorer

A simple web application to explore **Harry Potter characters** using an API.
Select a character from the dropdown and view detailed information instantly.

---

## 🚀 Live Demo

🔗 https://aryankumar62.github.io/REPO-NAME/

---

## 📌 Features

* 🔍 Search characters from dropdown
* 📖 View character details (name, house, actor, etc.)
* ⚡ Fast API-based data fetching
* 📱 Responsive design (mobile-friendly)
* ❗ Handles missing data (fallback values)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Fetch API
